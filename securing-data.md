# Data security
{: #mng-data}

{{site.data.keyword.openpages_full}} uses a Db2® database system to store data. Db2 encrypts data, both while in storage and in transit.

The database used to store data during the experimental testing phase for the {{site.data.keyword.openpages_full_notm}} service will be deleted from the database after the testing phase is complete. No testing data will be retained.
